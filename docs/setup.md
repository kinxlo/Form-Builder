# Setup

## Prerequisites

- Node.js (recommend the version you already use for Nuxt 4)
- Package manager: `pnpm` or whatever you prefer (npm/yarn)

## Install

```bash
pnpm install
```

## Run the app

```bash
pnpm dev
```

Open:

- http://localhost:3000
